# 🍏 تشغيل محول المستندات الذكي على أجهزة ماك (macOS)

الكود المصدري للبرنامج متوافق تماماً مع أجهزة ماك (Mac)، ولكن بما أن أجهزة ماك لا تدعم ملفات `.exe` الخاصة بنظام ويندوز، فلا يمكن استخدام المجلد المحمول (`dist`) مباشرة. يجب تشغيل البرنامج من الكود المصدري بعد تثبيت المتطلبات الخاصة بنظام ماك.

فيما يلي الخطوات السريعة لتشغيل البرنامج على أي جهاز ماك:

---

## الخطوة 1: تثبيت مدير الحزم Homebrew (إذا لم يكن مثبتاً)
افتح موجه الأوامر (Terminal) في الماك والصق هذا الأمر ثم اضغط Enter:
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## الخطوة 2: تثبيت محرك Tesseract واللغة العربية
أجهزة ماك تحتاج إلى تثبيت محرك Tesseract بشكل منفصل. في الـ Terminal، اكتب:
```bash
brew install tesseract
brew install tesseract-lang
```
*(هذا الأمر سيقوم بتنزيل المحرك وجميع اللغات بما فيها العربية `ara` والإنجليزية `eng`).*

## الخطوة 3: تثبيت Python (إذا لم يكن مثبتاً)
```bash
brew install python
```

## الخطوة 4: تشغيل البرنامج
1. قم بفك ضغط مجلد المشروع الأصلي (الكود المصدري وليس مجلد `dist`).
2. افتح الـ Terminal وتوجه إلى مجلد المشروع:
```bash
cd /مسار/المجلد/الخاص/بالمشروع
```
3. قم بإنشاء بيئة وهمية (اختياري ولكن يُنصح به):
```bash
python3 -m venv venv
source venv/bin/activate
```
4. قم بتثبيت مكتبات بايثون المطلوبة:
```bash
pip3 install -r requirements.txt
```
5. **شغّل البرنامج:**
```bash
python3 main.py
```

---

### 💡 ملاحظات هامة للماك:
- البرنامج سيتعرف تلقائياً على أنك تستخدم نظام ماك وسيبحث عن محرك Tesseract المدمج في النظام بدلاً من البحث عن ملف `tesseract.exe`.
- الكود البرمجي الذي كتبناه للتغلب على مشكلة المسافات في الويندوز (Monkey Patch) مصمم ليعمل على الويندوز فقط، ولن يتعارض أبداً أو يؤثر على عمل البرنامج في نظام ماك أو لينكس.
- سرعة التحويل على أجهزة ماك (خاصة بمعالجات M1/M2/M3) ستكون خيالية وسريعة جداً.
