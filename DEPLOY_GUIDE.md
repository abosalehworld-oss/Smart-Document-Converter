# 📦 دليل نقل البرنامج للأجهزة السيادية
# Transfer Guide - Secure Government Deployment

## ✅ ما تحتاجه: 3 ملفات فقط

---

## الطريقة الأولى: نقل المشروع كاملاً (للمطوّر)

### على جهاز فيه إنترنت:
1. تأكد إن `build_exe.bat` خلص وظهر مجلد `dist\SmartDocConverter\`
2. المجلد ده جاهز للنقل فوراً

### النقل:
```
انسخ مجلد dist\SmartDocConverter\ كامل على USB
```

### على الجهاز السيادي:
```
1. الصق المجلد في أي مكان (مثلاً C:\Programs\SmartDocConverter\)
2. اضغط دبل كليك على SmartDocConverter.exe
3. خلاص - لا Python - لا تثبيت - لا إنترنت
```

---

## الطريقة الثانية: نقل المشروع المصدري (للمطوّر التقني)

### انسخ هذه المجلدات فقط:
```
بروجكت برنامج OCR\
├── app\               ← الكود الأساسي
├── tests\             ← الاختبارات  
├── venv\              ← المكتبات المثبتة (مهم!)
├── models\            ← نماذج OCR (مهم!)
├── main.py
├── requirements.txt
├── run.bat            ← لتشغيل البرنامج
└── build_exe.bat      ← لبناء exe جديد
```

### ⚠️ لا تنسَ:
- مجلد `venv\` كامل (المكتبات)
- مجلد `models\` كامل (نماذج OCR)
- بدونهم البرنامج لن يعمل أوفلاين

---

## 🔒 تأكيدات الأمان قبل النقل

### على جهاز المصدر - نفّذ:
```
venv\Scripts\python security_check.py
```

### ما يجب التحقق منه:
- [ ] لا يوجد اتصال شبكة عند التشغيل
- [ ] مجلد `models\` يحتوي على ملفات .pth
- [ ] مجلد `venv\` موجود وكامل
- [ ] ملف `settings.json` لا يحتوي على بيانات حساسة

---

## 🚨 قواعد الاستخدام على الأجهزة السيادية

1. ❌ لا تفتح البرنامج وأنت متصل بالإنترنت
2. ❌ لا تسحب ملفات من خارج الشبكة الداخلية
3. ✅ الملفات الناتجة (.docx) احفظها على الشبكة الداخلية فقط
4. ✅ امسح الملفات المؤقتة من %TEMP% دورياً
5. ✅ لا تشارك مجلد venv\ مع أحد (فيه المكتبات كاملة)
