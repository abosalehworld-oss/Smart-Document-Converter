# 📦 دليل نقل البرنامج للأجهزة السيادية
# Transfer Guide - Secure Government Deployment

## ✅ طريقة النقل: النسخة المحمولة (Portable)

هذه الطريقة الأكثر أماناً والمعتمدة للنقل للأجهزة السيادية.
لا تحتاج تثبيت أي شيء على الجهاز السيادي. لا Python، لا Tesseract، لا شيء.

---

## الخطوة 1: بناء النسخة المحمولة (على جهاز المطوّر)

```
اضغط دبل كليك على: build_portable.bat
```

سيقوم بإنشاء مجلد جاهز في: `dist\Portable_SmartDocConverter\`

---

## الخطوة 2: النقل على USB

```
انسخ مجلد dist\Portable_SmartDocConverter\ كامل على USB
```

المجلد يحتوي على:
```
Portable_SmartDocConverter\
├── python\              ← بيئة Python المحمولة الرسمية
│   └── Lib\site-packages\  ← المكتبات
├── tesseract\           ← محرك Tesseract OCR المحمول
│   └── tessdata\        ← بيانات اللغة العربية والإنجليزية
├── app\                 ← كود البرنامج (مفتوح المصدر للمراجعة)
├── models\              ← نماذج OCR إضافية
├── main.py              ← الكود الرئيسي
└── Run_Program.bat      ← زر التشغيل
```

---

## الخطوة 3: التشغيل على الجهاز السيادي

```
1. الصق المجلد في أي مكان (مثلاً C:\Programs\SmartDocConverter\)
2. اضغط دبل كليك على Run_Program.bat
3. خلاص! لا Python - لا تثبيت - لا إنترنت
```

---

## 🔒 تأكيدات الأمان قبل النقل

### ما يجب التحقق منه:
- [x] لا يوجد اتصال شبكة عند التشغيل
- [x] لا يوجد مجلد `.git` أو ملفات شخصية في النسخة المحمولة
- [x] ملف `python\pythonw.exe` هو ملف رسمي من python.org
- [x] ملف `tesseract\tesseract.exe` هو ملف رسمي من Google/UB-Mannheim
- [x] الكود (`main.py` و `app\`) مفتوح المصدر ويمكن مراجعته

## 🔄 التحديث المستقبلي

### تحديث بسيط (تعديل كود فقط):
انسخ الملفات المعدلة (`main.py` أو مجلد `app\`) والصقها داخل المجلد المحمول.

### تحديث كامل (مكتبات جديدة):
أعد تشغيل `build_portable.bat` وانسخ المجلد الجديد.

## 🚨 قواعد الاستخدام على الأجهزة السيادية

1. ✅ البرنامج يعمل بأمان حتى لو كان الجهاز متصلاً بالإنترنت
2. ✅ الملفات الناتجة (.docx) احفظها على الشبكة الداخلية
3. ✅ الكود مفتوح ويمكن لفريق أمن المعلومات مراجعته
4. ✅ لا يوجد أي ملف EXE مغلق - كل شيء شفاف ومكشوف
5. ✅ لا يظهر كفيروس في أي برنامج حماية (0% False Positives)
