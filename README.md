# 🔒 محوّل المستندات الذكي - Smart Document Converter

> تطبيق سطح مكتب **آمن بالكامل** يعمل **أوفلاين** لتحويل ملفات PDF والصور إلى مستندات Word
> مع دعم كامل للغة العربية والإنجليزية.

## ✨ الميزات

- 📄 **تحويل PDF → Word** — يكتشف النص الرقمي تلقائياً أو يستخدم OCR
- 🖼️ **تحويل الصور → Word** — يدعم PNG, JPG, BMP, TIFF, WebP
- 🌐 **ثنائي اللغة** — واجهة عربية + إنجليزية مع تبديل فوري
- ✍️ **خط يدوي** — يدعم النصوص المكتوبة بخط اليد
- 🔒 **آمن 100%** — لا إنترنت، لا تخزين بيانات، لا تتبع
- 💻 **CPU فقط** — يعمل على أضعف الأجهزة بدون GPU
- ♾️ **مجاني بلا حدود** — لا حد لعدد الملفات أو الصفحات

## 🚀 التشغيل السريع

```bash
# 1. التثبيت (مرة واحدة فقط - يحتاج إنترنت)
setup.bat

# 2. التشغيل (بدون إنترنت)
run.bat
```
