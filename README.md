# 🔒 محوّل المستندات الذكي - Smart Document Converter

> تطبيق سطح مكتب **آمن بالكامل** يعمل **أوفلاين** لتحويل ملفات PDF والصور إلى مستندات Word
> مع دعم كامل للغة العربية والإنجليزية.

## ✨ الميزات

- 📄 **تحويل PDF → Word** — يحافظ على تنسيق وهيكل المستند الأصلي بدقة (سطر بسطر)
- 🖼️ **تحويل الصور → Word** — يدعم PNG, JPG, BMP, TIFF, WebP
- 🌐 **ثنائي اللغة** — واجهة عربية + إنجليزية مع تبديل فوري
- ✍️ **خط يدوي** — يدعم النصوص المكتوبة بخط اليد
- 🔒 **آمن 100%** — لا إنترنت، لا تخزين بيانات، وحماية من ملفات PDF والصور الخبيثة أو العملاقة
- 💻 **CPU فقط** — يعمل على أضعف الأجهزة بدون GPU مع إدارة ذكية للذاكرة
- ♾️ **مجاني بلا حدود** — لا حد لعدد الملفات أو الصفحات (تم التطوير لوجه الله تعالى)
- ⚡ **سريع جداً** — يعالج الصفحة الواحدة في 2-3 ثوانٍ

## 🚀 التشغيل السريع

```bash
# 1. التثبيت (مرة واحدة فقط - يحتاج إنترنت)
setup.bat

# 2. التشغيل (بدون إنترنت)
run.bat
```

## 📦 بناء النسخة المحمولة (للأجهزة السيادية)

```bash
# يقوم بإنشاء مجلد محمول جاهز للنقل على USB
build_portable.bat
```
النسخة المحمولة ستكون في: `dist\Portable_SmartDocConverter\`
